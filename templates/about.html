Skill Rank

Skill Rank

Skill Rank scours job postings from the web (Indeed.com) and uses a sophisticated ranking system to tell you the most relevant skills required for a particular position.

How it works - math alert!

Skill Rank uses an augmented TF-IDF (term frequency-inverse document frequency) approach to identify key words in a particular job title query. Enough jargon... Skill Rank maintains a large database of terms scraped from thousands of online job postings and how frequently those terms appear. This serves as background reference data for your job title query. When you perform a query, Skill Rank tracks the terms from those particular postings and how frequently they appear as well. From there, the relevance of a particular term resulting from your query is determined by the number of times it occurs (on average) in your query vs. the number of times it occurs (on average) in the background reference data set.

If a word is extremely common in general (e.g. "and"), and is equally common in the particular query, it will receive a low relevance score. However, if a word is uncommon in the reference database, but fairly common for the particular query, it will receive a higher relevance score. The more reference data Skill Rank possesses, the more powerful the algorithm becomes.
